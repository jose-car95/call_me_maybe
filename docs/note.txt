
------ Poceso de generacion del LLM: ------------
Prompt -> Tokenization -> Input IDs -> LLM -> Logits -> Next Token Selection


Logits: El modelo produce una distribucion de probabilidad(logits), sobre
    todos los posibles tokens siguientes

1- LLM:
LLM
    Un LLM es un modelo que predice el siguiente token.
    No “piensa” como una persona ni ejecuta código.
    Recibe texto convertido en números y devuelve puntuaciones sobre cuál podría ser el siguiente token.
